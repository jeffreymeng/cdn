The Jeffrey CDN

The Jeffrey CDN hosts content for sites spanning jeffkmeng.com and it's subdomains.
It provides a quick and centralized way to deploy resources. There are a few Jeffrey services that document public links to this CDN, some of the more popular are:

  • assets.jeffkmeng.com for code and programming libraries.
  • The Jeffrey Archives for all kinds of content.
